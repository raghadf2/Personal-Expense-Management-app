#  Personal Expense Manager App

A simple and interactive web application for managing personal expenses.  
The app allows users to add expenses, view the total amount, and delete items.  
Data is stored locally using browser localStorage, so information remains saved even after refreshing the page.

---

##  Features
- Add new expenses (description + amount)
- Delete expenses
- Automatically calculate total expenses
- Highlight expensive items (optional styling)
- Data persistence using localStorage

---

##  Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
- localStorage API

---

##  Purpose of the Project
This project was built to practice:
- DOM manipulation
- Event handling in JavaScript
- Working with arrays and objects
- Using localStorage for data persistence
